Inaccessible Components

Hover Effects

- Hover effects are used to indicate that an element can be clicked - And usually involve changing a colour - ANd we can see here, the colour is changing on hover, which means this must be clickable
- But what if the user has a vision deficiency, and is unable to differentiate between colours as well - Can you see the difference between the colours?
- Using something other than a colour change - Like changing a background, border or the underline
- No longer relies on differentiating between colours - So users can more easily tell the difference - Although I'd only recommend using the border as an option if the element has a bit of space around it
