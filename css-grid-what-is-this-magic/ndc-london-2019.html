- So we discovered that you could use tables for layouts
- This was great because you could break the apge into a grid
- You could merge cells into each other
- And embed tables inside tables
- We finally had better control over how content was laid out on a page
- But before long, we ended up with excessive complexities
- Tables inside tables was great
- But tables inside tables inside tables inside tables
- Not so much
- This also made it more difficult when we started trying to make out websites work on different device sizes
